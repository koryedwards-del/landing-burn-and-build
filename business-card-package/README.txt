Burn & Build — Business Card Package
=====================================

Contents
--------
  business-card-mockup.html   Open in browser — front + back at print size (1050×600 px each)
  business-card-editor.html   Click any text to edit (saves in browser)
  css/business-card.css       All layout and typography
  img/brand/bblogo.png        Square Burn & Build logo
  img/brand/qr-burnandbuilddiet.png   QR code → https://burnandbuilddiet.com

Quick start
-----------
  1. Unzip this folder anywhere on your computer.
  2. Double-click business-card-mockup.html (or open it in Chrome).
  3. Print: Ctrl/Cmd+P → 100% scale → turn OFF “Fit to page”.

Card size
---------
  3.5″ × 2″ standard US business card @ 300 DPI = 1050 × 600 pixels per face.
